Programming Assignment!

Welcome to this single-page application, built with:

  • ASP.NET Core and C# for cross-platform server-side code
  • Angular and TypeScript for client-side code
  • Bootstrap for layout and styling

Features showcased

  • Client-side navigation. For example, click Makelaars then Back to return here.
  • Angular CLI integration. Auto page refresh
  • Use of httpclient to consume the api
  • C# unit testing

Possible Functionalities

  • Get all the makelaars in a particular city in the Netherlands
  • Get the list of top 10 makelaars (those with the most properties for sale) in the chosen city
  • Get the list of top 10 makelaars (those with the most properties with a tuin for sale) in the chosen city
